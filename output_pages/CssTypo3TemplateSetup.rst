============================
TS/CSS: Typo3 Template Setup
============================

Using the Template module, go to your "root" page containing the root
template and select "Info / Modify" from the top-right drop-down box,
then "Click here to edit whole template record". Scroll down and:

-  Section "Clear", check: Constants, Setup
-  Check Section "Rootlevel"
-  Section "Include static (from extensions)", add: CSS Styled Content
   (called: content.static (default) )

| 
| (Make sure you have the CSS Styled Content extension installed
  already)
