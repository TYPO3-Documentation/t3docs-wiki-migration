==========================
TYPO3 Exception 1283192644
==========================

.. note::

Adding a TemplaVoila's datastructure via plugin (using
$GLOBALS['TYPO3_CONF_VARS']['EXTCONF']['templavoila']['staticDataStructures'][]
= $myDS;), this exception will be thrown if $myDS is null.
