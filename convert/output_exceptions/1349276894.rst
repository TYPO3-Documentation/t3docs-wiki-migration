==========================
TYPO3 Exception 1349276894
==========================

.. note::
   If you encountered this exception, please help others by providing
   information about how you got this error. Especially if you have a solution,
   please add it to this page by following the :ref:`"Edit on GitHub" workflow
   <docs-contribute-github-method>`!

This error most likely occurs because the current backend-user does not
have access to any file-mounts. Check that a file-mount exists and that
the current user has access to it - either through the group(s) or
through the user directly.
