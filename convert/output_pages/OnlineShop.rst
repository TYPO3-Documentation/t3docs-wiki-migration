Extensions based on tt_products
-------------------------------

The following extensions are based on tt_products: They extend its
functionality in one way or the other. Please note that most of these
extensions are quite old and/or no longer actively developed, since
tt_products has adopted some of the features in its more recent
releases.

-  zk_products
-  mkl_products
-  tx_twproducts
-  ast_rteproducts
-  onet_ttproducts_rte

Other shop extensions
=====================

-  `kb_shop <https://docs.typo3.org/typo3cms/extensions/kb_shop/>`__
-  `pt_gsashop <https://docs.typo3.org/typo3cms/extensions/pt_gsashop/>`__
   - a powerful shop system with ERP integration (GS Auftrag
   Professional), still under development
-  `meta_oscommerce <https://docs.typo3.org/typo3cms/extensions/meta_oscommerce/>`__
   - integrates osCommerce into TYPO3
-  `wt_cart <https://docs.typo3.org/typo3cms/extensions/wt_cart/>`__ -
   easy and flexible shopping extension
-  `Quick Shop <https://extensions.typo3.org/extension/quick_shop/>`__ -
   focused on easy setup and few products
-  `Multishop <https://extensions.typo3.org/extension/multishop/>`__ is
   a highly praised newcomer in the Typo3 shop scene, offering easy
   setup, multiple shops in one page tree and frontend editing
